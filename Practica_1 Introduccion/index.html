Ejercicios con problemas a resolver

Cumpleaños

Imagina que estás organizando una fiesta de cumpleaños para tu amigo y necesitas hacer una lista de los invitados, los regalos y la comida que vas a servir.

Para llevar a cabo esta tarea, necesitas declarar varias variables en tu mente o en papel para ayudarte a organizar la información de una manera más eficiente. Por ejemplo:

  1. Crea una variable llamada cumpleañero para saber como se llama.
  2. Crea una variable llamada invitado para llevar un registro de quién haz invitado a la fiesta.
  3. Crea una variable llamada regalo para llevar un registro del regalo que planeas dar a tu amigo.
  4. El mejor amigo del cumpleañero se llama Gaston, crea una condicion para revisar si gaston esta es nuestra variable invitado
  5. En el caso de cumplirse la condicion imprimir con un Alert el mensaje "Gaston esta en la fiesta."
  6. En el caso de no estar en nuestra variable invitado imprimir con un Alert el mensaje "Gaston no fue invitado."
  7. Nuestro cumpleañero se llama Luis, crea una condicional que revise si el nombre del cumpleañero es "Luis".
  8. En el caso de cumplirse la condicion imprimir con un Alert el mensaje "Luis recibio el variableRegalo."
  9. En el caso de no cumplirse nuestra condicion imprimir con un Alert el mensaje "Esta fiesta es para luiss.."

    10. Aclaracion: Usando el metodo prompt podras asignar valores a las variables creadas
